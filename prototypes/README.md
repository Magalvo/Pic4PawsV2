# Exploratory Prototypes

This folder preserves code created before the approved architecture proposal and SDD.

- `vite-visual-slice`: initial React/Vite visual prototype based on the supplied design references.
- `exploratory-api`: initial Express API prototype with mocked domain data.

These prototypes are not part of the approved production architecture. They can be referenced for visual direction or implementation ideas, but production work should proceed through `apps/mobile`, `apps/web`, `apps/workers` and the shared packages under `packages/`.
