# Design System

## Source References

The first remake UI is based on `docs/design-refs/stitch_dual_onboarding`.

## Personality

Pic4Paws should feel optimistic, urgent and community-centric. The app is mobile-first and action-led, with pet photography as the primary emotional surface.

## Tokens

- Primary: Rescue Orange `#ec5b13`
- Secondary: Teal `#2aa7a2`
- Background: `#f8f6f6`
- Surface: `#ffffff`
- Text: Slate-like ink `#0f172a`
- Muted text: `#64748b`
- Border: `#f1d8cd`
- Font: Inter, system fallback

## UI Rules

- Keep primary media at a 4:5 portrait ratio in feed cards.
- Use fixed-width mobile composition centered on desktop.
- Prefer direct actions over explanatory copy.
- Persistent bottom navigation should stay compact and icon-led.
- Pet detail screens should show adoption status, shelter, sponsorship progress and medical status at a glance.
